DeletionMode

enum class DeletionMode

Enum Values

ALWAYS

Always delete the created file after the test has run.

IF_SUCCESSFUL

Delete the file if the test ran through successfully; retain the file if the test failed. Retained files will be deleted the next time the test is executed.

NEVER

Retain the file after the test ran. Retained files will be deleted the next time the test is executed.
